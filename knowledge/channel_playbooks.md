# 渠道内容模板库

## X

### 单帖公式

- 钩子
- 一个痛点
- 一个方法观点
- 一个 CTA

示例结构：

1. 多数交易者不是缺策略，而是缺执行纪律。
2. 真正拉开差距的，不是预测，而是复盘。
3. 我们把复盘流程做成了可训练的系统。
4. 想看演示可以私信或预约试用。

### 线程公式

1. 反常识开头
2. 3-5 个核心观点
3. 一个方法总结
4. CTA

## 小红书

### 标题公式

- 我是怎么开始认真做交易复盘的
- 交易久了才发现，最重要的不是买点
- 如果你总是知行不一，先别急着换策略

### 正文公式

1. 个人观察/常见误区
2. 为什么会这样
3. 一个可执行方法
4. 工具或系统如何辅助
5. 结尾 CTA

## 微信公众号

### 长文结构

1. 开头问题
2. 用户常见误区
3. 方法框架
4. 场景案例
5. 工具如何辅助执行
6. 风险提醒
7. CTA

## 雪球

### 观点帖结构

1. 一个交易认知观点
2. 为什么很多人做不到
3. 一个复盘或风控框架
4. 互动问题

### 长帖结构

1. 复盘背景
2. 当时的判断依据
3. 哪些地方做对了
4. 哪些地方做错了
5. 下次如何修正
6. CTA
